Финальный чеклист BASIC-5.

Этот файл хранит текстовый вариант вывода команд. Для сдачи руководителю нужен скриншот терминала Ubuntu в WSL.

Команды для проверки:

docker --version
docker compose version
docker run hello-world
uv --version
uv python install 3.12
uv run --with fastapi python -c "import fastapi; print('FastAPI', fastapi.__version__, 'OK')"
node --version
npm --version
code --version
