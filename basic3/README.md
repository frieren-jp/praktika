# BASIC-3 - подготовка к ревью ToDo API

В этой папке лежат материалы для подготовки к живому демо и code review проекта BASIC-2.

Файл:

- `review-prep.md` - план демо, ответы на ожидаемые вопросы и рефлексия.

Перед встречей:

1. Открыть `review-prep.md`.
2. Прогнать запуск:

```powershell
cd C:\Users\freezemyself\Desktop\praktika\todo_app
.\check.ps1
```

3. Для Swagger-демо запустить:

```powershell
.\run.ps1
```

И открыть `http://127.0.0.1:8000/docs`.
